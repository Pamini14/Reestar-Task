Reestar-Task HTML & CSS

Project Overview :
This project is a simple and clean Food Menu Card created using basic HTML and CSS. It displays food items in an organized layout with proper structure and styling. The design is easy to understand and beginner-friendly.

Features:
Simple and neat layout
Proper use of HTML elements
Basic CSS styling
Clean fonts and spacing
Well-organized menu sections

Technologies Used:
HTML5 – For creating the structure of the menu
CSS3 – For basic styling such as colors, fonts, margins, padding, and borders

Purpose:
This project was built to practice fundamental HTML and CSS concepts. It is suitable for beginners who are learning web design and want to understand basic webpage structure and styling.
